Examples

See QAIL in action with real-world query examples.

Basic Queries

Select all users get::users:'_
Generated SQL SELECT * FROM users
Select specific columns get::users:'id'email'name
Generated SQL SELECT id, email, name FROM users

Filtering

Filter by condition get::users:'_ [ 'active == true ]
Generated SQL SELECT * FROM users WHERE active = true
Fuzzy search get::users:'id'name [ 'name ~ "john" ]
Generated SQL SELECT id, name FROM users WHERE name ILIKE '%john%'

Sorting & Pagination

Sort descending get::users:'_ [ -created_at ]
Generated SQL SELECT * FROM users ORDER BY created_at DESC
Pagination with range get::users:'_ [ 20..30 ]
Generated SQL SELECT * FROM users LIMIT 10 OFFSET 20

Combined Example

Full query with filter, sort, and pagination get::users:'id'email [ 'active == true, -created_at, 0..10 ]
Generated SQL SELECT id, email FROM users WHERE active = true ORDER BY created_at DESC LIMIT 10
